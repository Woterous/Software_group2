# Iteration Plan

## Overview

The project will be developed using Agile methodology with multiple iterations.

Each iteration focuses on a different stage of development.

---

## Iteration 1

Focus:

- requirement analysis
- writing user stories
- creating product backlog
- system architecture design
- prototype design

This stage focuses on understanding the system and planning development.

---

## Iteration 2

Focus:

- user registration
- login system
- profile management
- job posting
- job browsing
- job application

This iteration will produce the first working version of the system.

---

## Iteration 3

Focus:

- application management
- admin functions
- workload monitoring
- system improvements

This iteration improves functionality and usability.

---

## Iteration 4

Focus:

- system testing
- bug fixing
- performance improvements
- final documentation
- demonstration preparation
