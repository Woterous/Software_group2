# User Stories

## Teaching Assistant

As a Teaching Assistant, I want to register an account so that I can apply for TA jobs.

As a Teaching Assistant, I want to log into the system so that I can access my account.

As a Teaching Assistant, I want to create a personal profile so that module organisers can understand my skills.

As a Teaching Assistant, I want to upload my CV so that I can show my qualifications.

As a Teaching Assistant, I want to browse available jobs so that I can find suitable opportunities.

As a Teaching Assistant, I want to view job details so that I understand the job requirements.

As a Teaching Assistant, I want to apply for a job so that I can become a TA.

As a Teaching Assistant, I want to check my application status so that I know whether I was selected.

---

## Module Organiser

As a Module Organiser, I want to post TA jobs so that I can recruit teaching assistants.

As a Module Organiser, I want to view applicants so that I can evaluate candidates.

As a Module Organiser, I want to review applicant skills so that I can select suitable candidates.

As a Module Organiser, I want to select applicants so that I can assign TAs to my module.

---

## Admin

As an Admin, I want to view all users so that I can manage system users.

As an Admin, I want to view all job applications so that I can monitor the recruitment process.

As an Admin, I want to check TA workload so that tasks are fairly distributed.
