# TA UI Overhaul Note

This refresh focuses on reducing the "AI template" feel without changing the existing TA workflow structure.

## What Changed

- Rebuilt the visual system with a calmer, more mature palette based on deep teal and warm neutrals instead of generic bright dashboard blue.
- Introduced differentiated surface hierarchy so the top bar, sidebar, hero areas, stats, filters, cards, tables, and upload panels no longer share the same visual weight.
- Reduced uniformity by varying panel density, card emphasis, spacing rhythm, and content layout rather than treating every module as an identical rounded rectangle.
- Reworked TA Overview into a real summary hero plus weighted stats and more product-like activity modules.
- Turned My Profile into a settings-style page with a supporting readiness rail and a clearer CV status card.
- Rebuilt Job Board cards around title, deadline urgency, workload, and skills so scanning feels intentional and task-driven.
- Refined Applications into a more product-like data table with stronger row structure, helper metadata, and clearer status context.

## How The "AI Feel" Was Reduced

- Broke the repeated "same border, same shadow, same radius, same emphasis" pattern across the app.
- Replaced generic blue gradient CTAs with a more restrained action hierarchy and less templated button styling.
- Added context copy, helper text, deadline urgency cues, empty states, and activity notes so the interface feels used and maintained rather than auto-generated.
- Strengthened navigation hierarchy through richer sidebar item structure and a more editorial top shell.
- Kept motion subtle and fast so feedback feels intentional, not decorative.

## Result

The interface should now read more like a focused campus recruitment SaaS product and less like a uniform coursework dashboard mockup.
